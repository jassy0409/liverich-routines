# Project instructions

## Design system

All UI, web pages, and visual work in this project must follow the Apple
Design Pack in `design/apple-design-pack.md`. Read that file before doing any
design work. Key points:

- Typography first: SF Pro stack, headlines semibold (600) or bolder,
  negative letter-spacing on large type.
- Colors: light #FFFFFF / #F5F5F7, dark #000000 / #1D1D1F, text #1D1D1F /
  #6E6E73 (light) and #F5F5F7 / #D2D2D7 (dark), accent blue #0066CC
  (#2997FF on dark).
- Pill buttons (border-radius 980px), cards with 28–32px radius, concentric
  nested corners.
- No chips/badges; hierarchy from type weight and size, not decoration.
- Generous asymmetric section padding, reveal-on-scroll and hover-scale
  motion.

Before shipping any screen, run the review checklist in Part 10 of the
design pack.
